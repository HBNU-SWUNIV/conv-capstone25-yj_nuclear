# 한밭대학교 컴퓨터공학과 영&재는 영재야팀

**팀 구성**
- 20222011 최희재
- 20222042 장영아

## <u>Teamate</u> Project Background
- ### 필요성
  - 플랜트, 발전소와 같은 대형 산업 설비에서는 유체기기가 안정적으로 동작해야 한다.
    - 특히 제어밸브의 안정적인 동작은 플랜트 안전운영의 핵심 요소이다.
  - 밸브를 구동하는 공압구동기는 실제 밸브를 개폐하는 중추적 역할을 한다.
    - 그러나 반복적인 개폐, 압력 변화, 고온 등의 환경에 지속적으로 노출되며, 비금속 재료인 O-Ring 등의 열화와 마찰력 증가는 성능저하와 고장의 주요 원인으로 보고된다.
- ### 기존 해결책의 문제점
  - Auto-Calibration : 기존 연구의 자가진단 기법에서는 추세를 예측하기는 힘들다.
  
## System Design
  - ### System Requirements
    - 센서를 통해 시뮬레이션에 필요한 데이터셋을 확보
    - 데이터 전처리를 통해 노이즈 제거 및 이상치 제거
    - 머신러닝 기반 상태분류 및 성능추세 예측 모델 개발
    - 모델의 설명가능성을 확보하여 현장 적용 신뢰성 강화
  
## Conclusion
  - ### 상태기반정비 체계를 가능하게 하여, 부품 교체 시기를 최적화하고 불필요한 유지보수 비용을 줄인다.
  - ### 향후 스마트 플랜트, 예지정비 분야와 연계할 수 있는 기술적 차별성을 제공한다.
